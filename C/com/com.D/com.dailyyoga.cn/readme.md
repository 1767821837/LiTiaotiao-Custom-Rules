## 普通规则

快速复制:
```
{"popup_rules":
    [
        {"id":"新人专属福利","action":"iv_close"},
        {"id":"hbv","action":"iv_close"}
    ]
}
```
详细说明：
- [{"id":"新人专属福利","action":"iv_close"}](#id新人专属福利actioniv_close)
- [{"id":"hbv","action":"iv_close"}](#idhbvactioniv_close)
