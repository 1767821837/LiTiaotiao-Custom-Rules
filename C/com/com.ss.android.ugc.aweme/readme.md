## 去广告规则

快速复制:
```
{"popup_rules":
    [
        {"id":"boa","action":"boa"},
        {"id":"朋友推荐","action":"close"}
    ]
}
```
详细说明：
- [{"id":"boa","action":"boa"}](#idboaactionboa)
- [{"id":"朋友推荐","action":"close"}](#id朋友推荐actionclose)

### {"id":"boa","action":"boa"}
去除首页青少年模式弹窗

### {"id":"朋友推荐","action":"close"}
去除我的页面朋友推荐弹窗
