## 去广告规则

快速复制:
```
{"popup_rules":
    [
        {"id":"发现新版本","action":"cancel"},
        {"id":"moreIconRight","action":"moreIconRight"}
    ]
}
```
详细说明：
- [{"id":"发现新版本","action":"cancel"}](#id发现新版本actioncancel)
- [{"id":"moreIconRight","action":"moreIconRight"}](#idmoreiconrightactionmoreiconright)

### {"id":"发现新版本","action":"cancel"}
去除更新弹窗提示

![](./assets/更新弹窗.jpg)

### {"id":"moreIconRight","action":"moreIconRight"}
